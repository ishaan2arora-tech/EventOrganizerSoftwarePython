This is an event organization software for many to use to plan events in a smooth, timely, and organized manner.
It uses nested lists to store the records of the event details and the customer details.
You can:
        ADD A NEW EVENT
        UPDATE A EVENT BY ID
        DELETE AN EVENT BY ID
        VIEW AN EVENT BY ID
        VIEW ALL EVENTS
        REGISTER/ADD A NEW CUSTOMER
        UPDATE A CUSTOMER BY ID
        DELETE A CUSTOMER BY ID
        VIEW A CUSTOMER BY ID
        VIEW ALL CUSTOMERS
        MAKE A RESERVATION FOR AN EVENT
        CANCEL RESERVATION FOR THE EVENT
        SHOW THE CUSTOMER LIST OF AN EVENT
